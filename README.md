# Customer Behavior Analysis

## Business Problem
The objective of this project is to analyze customer transaction and demographic data to identify high-value customers, understand churn behavior, and provide actionable insights for retention strategies.

## Dataset
Customer transaction and demographic dataset containing 500 customer records.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## Key Insights
- High-value customers contribute the majority of revenue.
- Customers with low tenure are more likely to churn.
- Behavioral metrics are stronger predictors than income alone.

## Business Recommendations
- Introduce loyalty programs for high-value customers.
- Target early-tenure customers with engagement campaigns.
- Use behavior-based segmentation for marketing strategies.
